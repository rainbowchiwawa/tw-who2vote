# 任務
請依照輸入，搜尋使用者所在選舉區於該年度的所有候選人以及擬候選人，依其事蹟生成「李克特量表」問卷題目。
# 需要搜尋哪些資料
- 該候選人的完整中文姓名。
- 該候選人所屬政黨的完整中文名稱。
- 該候選人的現任職位及參選狀態(黨內提名/已登記/媒體或輿論熱門人選/其他)。
- 該候選人的黑歷史(越嚴重越好，尤其是那種選民可能從未聽過，卻會大幅影響意願的冷知識)。
# 問卷的題目形式
- 以候選人分組，確保題目中不提及該候選人的名字以及他所屬的政黨，避免填答者因既定印象影響填答結果。
- 題目敘述請以「我曾因為」為開頭，使用該候選人的視角以簡短的一句話講述他的黑歷史。
- 如果有某些專有名詞或抽象的法案、概念，請以「平常不接觸政治議題的人」都能聽懂的形容來取代。
- 如果有提及某個特定事件，請以一句話簡短補充該事件的來龍去脈。
## 題目敘述範例(依嚴重程度高到低排序)
1. `我曾因為...(做了什麼事/說了什麼話)...直接/間接使得(某人)自殺/被殺害。`
2. `我曾因為...(做了什麼事/說了什麼話)...直接/間接使得(某個災害)發生。`
3. `我曾因為...(做了什麼事/說了什麼話)...遭到選民罷免。`
4. `我曾因為...(做了什麼事/說了什麼話)...被判刑/罰款。`
5. `我曾因為...(做了什麼事/說了什麼話)...被檢調立案偵查。`
6. `我曾因為...(做了什麼事/說了什麼話)...遭到輿論撻伐。`
# 注意事項
- 請使用Markdown格式。
- 請為每位候選人製作5個問題。
- 請確保每一個黑歷史都是事實並且有證據支撐。
- 候選人數量不限，由每個黨的提名、初選資訊、媒體預測以及輿論熱門人選等決定。
- 請確保每一位候選人皆具備該年度的參選資格，如「連任兩次不得連任」或「戶籍不在當地」者必須刪除。
- 請在每個問題的題目敘述後附上該題目相關事件的「搜尋關鍵字」，以供填答者事後查詢。
- 不需要列出選項，以節省Token。
