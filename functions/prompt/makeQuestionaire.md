# 任務
請依照輸入，搜尋使用者所在選舉區於該年度的所有候選人以及擬候選人，依其事蹟生成「李克特量表」問卷題目、並列出資料來源的URL。
# 需要搜尋哪些資料
- 該候選人的完整中文姓名。
- 該候選人所屬政黨的完整中文名稱。
- 該候選人的現任職位及參選狀態(黨內提名/已登記/媒體或輿論熱門人選/其他)。
- 該候選人的事蹟，曾經說過的話、曾經做過的事(尤其是那種選民可能從未聽過，卻會大幅影響意願的冷知識或黑歷史)。
# 問卷的題目形式
- 以候選人分組，但不要在題目中提及該候選人的名字，避免填答者因名字或政黨產生既定印象；如有需要，請以「這個人」替代。
- 簡潔扼要的整理問題，但請明確並具體寫出「做了什麼事」或「說了什麼話」；如果有某些專有名詞或抽象的法案、概念，請以三歲小孩能聽懂的形容來取代。
- 如果該候選人的主張與事實有落差、或著是個廢話，請移除這個問題。
- 多建立與該候選人黑歷史相關的題目，並在敘述中讓填答者明白其嚴重性。
# 注意事項
- 請使用Markdown格式。
- 請為每位候選人製作5個問題。
- 候選人數量不限，由每個黨的提名、初選資訊以及媒體預測等決定，切勿漏掉某些潛在候選人。
- 請確保每一位候選人皆具備該年度的參選資格，如「連任兩次不得連任」或「戶籍不在當地」者必須刪除。
- 請確保「同意題目敘述」即為「支持該候選人」(例如將「這個人做了爭議的事情，可能影響他的施政能力」改為「這個人做了爭議的事情，但我認為不影響其施政能力」)。
- 請在每個問題的題目敘述後附上該題目相關事件的「新聞標題」，以供填答者事後查詢。
- 請在問卷最後附上每一組對應的候選人姓名及政黨。
- 不需要列出選項，以節省Token。
