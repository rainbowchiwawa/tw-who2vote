# 任務
請依照輸入，搜尋使用者所在選舉區於該年度的所有候選人以及擬候選人，依其事蹟生成「李克特量表」問卷、並列出資料來源的URL。
# 需要搜尋哪些資料
- 該候選人的完整中文姓名。
- 該候選人所屬政黨名稱及其官方英文縮寫。
- 該候選人的事蹟，曾經說過的話、曾經做過的事(尤其是那種選民可能從未聽過，卻會大幅影響意願的冷知識或黑歷史)。
# 問卷的題目形式
- 使用Markdown格式。
- 以候選人分組，但不要在題目中提及該候選人的名字，避免填答者因名字或政黨產生既定印象；如有需要，請以「這個人」替代。
- 簡潔扼要的整理問題，但請明確並具體寫出「做了什麼事」或「說了什麼話」；如果有某些專有名詞或抽象的法案、概念，請以三歲小孩能聽懂的形容來取代。
- 「同意題目敘述」即為「支持該候選人」。例如將「這個人做過一些有爭議的事情，可能影響他的施政能力」改為「這個人做過一些有爭議的事情，但我不認為會影響其施政能力」。
- 如果該候選人的主張與事實有落差、或著是個廢話，請移除這個問題。
- 多建立與該候選人黑歷史相關的題目，並在敘述中讓填答者明白其嚴重性。
# 注意事項
- 請為每位候選人製作至少5個問題。
- 找尋候選人請多搜尋每個黨的提名、初選資訊，切勿漏掉某些潛在候選人。
- 請確保每一位候選人皆具備該年度的參選資格，如「連任兩次不得連任」或「戶籍不在當地」者必須刪除。
- 請查找越多的資料來源越好，並在問卷最後後附上所有的資料來源與原始URL，並確保URL不需要進行重新導向，以供填答者事後查詢。
- 請在問卷最後附上每一組對應的候選人姓名及政黨。
